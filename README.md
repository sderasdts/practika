
# Система рекомендаций по питанию (ML Project)

## Описание
Учебный проект по машинному обучению. Приложение анализирует блюда русской кухни и определяет, подходят ли они для конкретной диетической цели: похудения, набора массы или поддержания баланса. Если блюдо не подходит, система рекомендует более подходящую альтернативу из той же категории.

## Технологии
- **Python**: Pandas, NumPy (обработка данных)
- **Machine Learning**: Scikit-learn (Random Forest Classifier, StandardScaler)
- **Web Interface**: Streamlit (интерактивное веб-приложение)
- **Serialization**: Joblib (сохранение моделей)

## Структура проекта
- `app.py` — код веб-приложения (запуск через Streamlit).
- `проект.ipynb` — Jupyter Notebook с полным циклом работы: генерация данных, обучение, тесты.
- `dataset.csv` — исходный датасет (300+ уникальных рецептов).
- `pr_dataset.csv` — обработанные данные с целевыми метками.
- `russian_food_models.pkl` / `russian_food_scaler.pkl` — сохраненные модели и скалер.

## Как запустить приложение

1. Установите необходимые библиотеки:
   ```bash
   pip install pandas numpy scikit-learn streamlit joblib
   ```

2. Запустите веб-интерфейс:
   ```bash
   python -m streamlit run app.py
   ```
   Приложение откроется в браузере (обычно http://localhost:8501).

## Логика работы
1. **Датасет**: Синтетически сгенерированная база из +-300 блюд с реалистичными вариациями (КБЖУ, способ приготовления).
2. **Целевые переменные**: Созданы на основе статистики (медианы/перцентили):
   - *Похудение*: низкие калории + низкий сахар.
   - *Набор массы*: высокий белок + достаточная калорийность.
   - *Баланс*: средние значения калорийности.
3. **Модель**: Ансамбль Random Forest обучается отдельно для каждой цели.
4. **Рекомендации**: При несоответствии цели алгоритм ищет замену внутри категории (минимум калорий для похудения / максимум белка для массы).
